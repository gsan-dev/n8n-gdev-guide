# 13.3 Tabla de comandos útiles

## Introducción

Aquí se presenta una tabla con comandos útiles para trabajar con n8n.

| Comando | Descripción |
|---------|-------------|
| `n8n start` | Inicia el servidor de n8n. |
| `n8n export:workflow` | Exporta un flujo en formato JSON. |
